### Configurazioni:

Ubuntu:
- Installazione di docker e docker compose
- Assegnare un indirizzo ip statico al PI

Docker-compose:
- Heidmall
- PiHole
    - Server dns
    - Local dns resolver
- Portainer
- Traefik
    - Local reverse proxy
    - HTTPS con certificato autofirmato
- Wireguard
